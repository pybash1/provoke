# templates/

## Purpose

Contains the Jinja2 HTML templates for the Flask web application.

## Files

- **`index.html`**: The search interface.
- **`admin.html`**: The main dashboard with charts and statistics.
- **`domains.html`**: Detail view for pages indexed under a specific domain.
- **`lists.html`**: Interface for adding/removing items from the blacklist and whitelist.
- **`label.html`**: The manual labeling interface.

## Styling

Styling is typically embedded or linked within these templates. They use modern CSS features to provide a premium administrative experience.
